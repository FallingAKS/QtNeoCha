## 技术难点

- 尝试加入qrc文件之前，输出正常；在加入qrc文件后，字母无法输出。
  原因：Windows对文件大小写不敏感，但是qrc文件是敏感的。qt的keyevent中，字母均用大写表示，无论输入字母大写还是小写，监听到的事件都是大写字母。在找文件时，win系统可以找到，qrc就找不到了。
